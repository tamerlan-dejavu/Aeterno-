This text will be encoded as UTF-8 bytes and padded to 64 bytes using PKCS#7.
64-byte plaintext loaded into a 4×4×4 tensor.
Row-major order: tensor[x][y][z] = data[x*16 + y*4 + z]
Circular layer shifts applied: X-planes → Y-planes → Z-planes.
Each plane shifts by its derived shift value (derived from key hash).
Each cell is XORed with a byte from the 64-byte key-derived mask.
The mask is generated from SHA-256(key) using a PRNG seeded with bytes 12–19.
Final encrypted tensor. All 64 cells have been transformed.
Result is Base64 encoded for safe transmission.
